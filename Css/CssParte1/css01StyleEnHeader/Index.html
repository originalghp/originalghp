Declaraciones en bloque definidas entre etiquetas style

h1 {
font-size: 2em;/*"Una vez y medio lo definido para el contenedor padre"*/
color:darkblue;
text-align: center;
background-color: aquamarine;
}

Todas las etiquetas con h1 tienen este estilo

La etiqueta h1 lleva todo el ancho del contenedor (por default lleva display de bloque)

La etiqueta h1 lleva texto en colo azul

El background de h1 lleva color aquamarine

El estilo definido para la etiqueta h1 indica que el texto estara centrado

p{
text-align: left;
font-size: 1em;
color:blue;
font-style:italic>;
border-top-style:solid;
border-bottom-style:solid;
border-width:thin;
border-color: blue;
}

Todos los textos con etiquetas p tienen este estilo definido dentro de la etiqueta style

Para los párrafos se define un estilo con estas características: Alineación a la izquierda

Tamaño de la letra de la misma medidad que la definida para el contenedor body

Color de letra azul

Estilo de fuente itálico

Borde: superior e inferior sólidos;finos y de color azul
